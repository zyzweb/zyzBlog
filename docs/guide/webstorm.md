### 彻底删除webstorm

首先用腾讯柠檬卸载

```bash
#找到jetbrains开头的文件和文件夹
cd ~/Library/Application\ Support/JetBrains/  
cd ~/Library/Preferences/  
cd ~/Library/Logs/JetBrains
cd ~/Library/Caches/JetBrains
```
